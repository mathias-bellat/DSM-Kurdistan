# Soil information in Kurdistan region, Dohuk governorate (Iraq), code and supplementary file

This deposit contains the code and the HTML supplementary file to Bellat, M.; Glissman, B.; Rensthscler, T.; Sconzo, P.; Kakhani, N.; Kohsravani, P.; Brifkany, B.; Pfälzner, P.
and Scholten, T., 2025. *Soil properties in northern Kurdistan region, Dohuk province (Iraq)*.

We present here the full workflow process of the paper within a [Bookdown file]((https://mathias-bellat.github.io/)/DSM-Kurdistan/)  

This is a complete description of the worflow related to the paper *Soil information in Kurdistan region, Dohuk governorate (Iraq)*. It contains in height different steps all the pre-process and necessary data to understand the whole process. This presentation of the material follows the **[FAIR](https://www.nature.com/articles/sdata201618) (Findable, Accessible, Interoperable, and Reuseable)** principles.
